Library Management System (OOP & File Handling)
A robust Java-based management system designed to handle library operations. This project demonstrates the practical application of Object-Oriented Programming (OOP) principles and Data Persistence through file handling.

Key Features
Modular Architecture: The system is split into specialized classes and interfaces to ensure clean, maintainable code.
Interface-Driven Design: Implements Lendable, Returnable, and Displayable interfaces to define strict behavior for library assets.
Persistent Data Storage: Uses FileHandler.java to read from and write to LibraryData.txt, ensuring that book records are saved even after the program is closed.
Inventory Management: Efficiently search, add, and update the status of books within the library system.
